# Ripple AI

Ripple AI is a production-style Android AI assistant built using Kotlin and modern Android architecture.

## Features

- AI Chat with streaming responses
- Vision AI using ML Kit
- Voice-to-text AI interaction
- Retry & partial failure handling
- Firebase Analytics logging

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Coroutines + Flow
- Retrofit
- ML Kit
- Firebase Analytics
- Hilt Dependency Injection
